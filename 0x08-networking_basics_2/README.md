# Bash - Networking basics #1

## Description of the files inside this folder:
0. Bash script that configures an Ubuntu server. The checker is running on Docker. localhost resolves to 127.0.0.2. facebook.com resolves to 8.8.8.8.
1. Bash script that displays all active IPv4 IPs on the machine it’s executed on. The IPs displayed may be different depending on which machine you are running the script on.

## Languages and Tools:

- OS: Ubuntu 20.04 LTS
- Language: Bash script
- Style guidelines: [Shellcheck](https://github.com/koalaman/shellcheck)
