# Web stack debugging #2

## Description of the files inside this folder:

0. Bash script that accepts one argument and should run the whoami command under the user passed as an argument.
1. Bash script that configures the container to run nginx as nginx user and nginx must be listening on all active IPs on port 8080.

## Languages and Tools:

- OS: Ubuntu 20.04 LTS
- Language: Bash script
- Web Server: Nginx
- Style guidelines: [Shellcheck](https://github.com/koalaman/shellcheck)
